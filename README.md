# Resume Classification using MLOps

## Overview
This project automates resume classification using machine learning and CI/CD pipeline.

## Tech Stack
- Python
- Scikit-learn
- GitHub Actions

## Workflow
1. Push code to GitHub
2. GitHub Actions triggers pipeline
3. Model trains automatically
4. Accuracy is generated

## Model
- TF-IDF Vectorizer
- Logistic Regression

## How to Run
pip install pandas scikit-learn
python src/train.py

## Output
Displays model accuracy after training.

## Conclusion
Demonstrates automation of ML workflow using MLOps principles.
